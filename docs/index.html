Analisi automatica
Carica il file rvtools.xlsx e ottieni statistiche complete in secondi. Nessuna configurazione manuale.
Report HTML interattivi
Dashboard con statistiche VM: accese, spente, totali. Navigazione e filtri embedded nel report.
Export PDF con branding
Genera PDF con logo e colori aziendali. Layout ottimizzato per la stampa, powered by WeasyPrint.
Breakdown OS con drill-down
Raggruppamento intelligente dei sistemi operativi. Visualizzazione dettagliata con espansione per famiglia OS.
Impostazioni personalizzabili
Configura logo, colori e nome azienda dall'interfaccia. Ogni report riflette la tua identità.
Deploy systemd nativo
Service file incluso per deploy su Linux. Avvio automatico all'accensione, gestione completa con systemctl.

// installazione

Operativo in
5 minuti.

Nessuna dipendenza cloud, nessun account. Un server Linux, Python 3, e sei pronto.

01
Clona il repository
Clone da GitHub nella directory target. Struttura minimale, subito chiara.
02
Installa le dipendenze
Sistema + Python: apt per WeasyPrint/Pango, pip per Flask, pandas e il resto.
03
Avvia o configura systemd
Test manuale con gunicorn, oppure abilita il service file per avvio automatico.
04
Carica l'export e genera
Upload del file rvtools.xlsx, configura branding, esporta HTML o PDF.
Quick start
Systemd
Dipendenze
# Clone repo
git clone github.com/abeggi/rvtools-report
cd rvtools-report

# Installa requirements Python
pip3 install -r requirements.txt

# Crea cartella dati
mkdir -p rvtools_data

# Avvia l'applicazione
export DATA_DIR=$(pwd)/rvtools_data
cd app
gunicorn --bind 0.0.0.0:8080 app:app
# Copia il file di servizio
sudo cp rvtools-analyzer.service \
  /etc/systemd/system/

# Ricarica e abilita
sudo systemctl daemon-reload
sudo systemctl enable rvtools-analyzer
sudo systemctl start rvtools-analyzer

# Verifica stato
sudo systemctl status rvtools-analyzer

# Segui i log live
journalctl -u rvtools-analyzer -f
# Dipendenze sistema (Debian/Ubuntu)
sudo apt-get update && apt-get install -y \
  python3-pip python3-venv \
  build-essential python3-dev libffi-dev \
  libpango-1.0-0 libpangoft2-1.0-0 \
  libharfbuzz0b libpangocairo-1.0-0 \
  libglib2.0-0 libcairo2 libfontconfig1 \
  libjpeg-dev libopenjp2-7-dev \
  shared-mime-info fonts-liberation

# Nota: fonts-liberation richiesto
# per output PDF corretto (WeasyPrint)
